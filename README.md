
# 🎭 Automation Exercise - Playwright

This repository contains an end-to-end test automation project built with Playwright and TypeScript. It is based on the test cases from the site Automation Exercise, and follows best practices including Page Object Model (POM), utilities, and visual reporting.

## 📁 Tech Stack

•	Playwright 

•	JavaScript

•	Page Object Model

•	HTML report

•	Git Hub Action

## 📸 Reports

HTML reports are automatically generated after each run.

Traces and videos are retained on failure for easier debugging.
## 📁 Running Tests

To install dependencies 

```bash
  npm install
```

To run tests in headless mode

```bash
  npx playwright test
```

To Run tests in headed mode

```bash
  npx playwright test --headed
```
